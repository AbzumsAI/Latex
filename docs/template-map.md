# Template map

I use this map when I come back to the repo after a while.

- `Template\HW.tex` controls the order of the sheet.
- `Template\questions` holds problem text.
- `Template\answers` holds hint text.
- `Template\commons\course.sty` holds shared commands.
- `Template\fonts` holds local Persian fonts.
- `Template\HW.pdf` is the built sample PDF.

Most edits should touch one folder and one idea at a time.
